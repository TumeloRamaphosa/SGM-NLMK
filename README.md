## How to Host on GitHub Pages (Free)

1. Go to **github.com** → Sign in → Click **"New"** repository
2. Name it `studex-oracle`, set to **Public**, add README → **Create**
3. Click **"Add file" → "Upload files"** → drag in `index.html`
4. Commit changes
5. Go to **Settings → Pages → Branch: main → Save**
6. Your live URL: `https://[your-username].github.io/studex-oracle`

**Live within 1–2 minutes.**
